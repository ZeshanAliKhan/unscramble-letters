# Unscramble Letters

Unscramble Letters is a browser-based word finder that helps users turn mixed letters into real English words. It supports wildcard searches, quick filters, grouped results, and a local generated word list for fast lookups without a backend.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Test

```bash
npm run test
```

## Deploy notes

- Build command: `npm run build`
- Publish directory: `dist`
- Netlify uses the config in `netlify.toml`
